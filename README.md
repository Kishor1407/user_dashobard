# 🚀 User Dashboard Application
Working link - user-dashobard-12zxho6cu-kishor1407s-projects.vercel.app



A modern, dynamic, and highly interactive Angular-based User Dashboard application. This project demonstrates state-of-the-art web development practices, featuring real-time data management, persistent storage, and beautiful data visualizations.

---

## ✨ Features

### 📊 Dynamic Dashboard
- **Interactive Analytics**: Integrated with **Chart.js** to provide visual insights into user distributions.
- **Real-time Statistics**: Instantly reflects changes in the user base without page reloads.

### 👥 User Management
- **Intuitive Forms**: Clean and responsive user registration form with validation.
- **Role-based Categorization**: Assign roles like **Admin**, **Editor**, or **Viewer** to manage permissions.
- **Smart Filtering**: Search and filter through users efficiently.

### 💾 Data Persistence
- **LocalStorage Integration**: All user data is persisted locally in the browser, ensuring your data remains available even after a refresh.
- **RxJS State Management**: Uses `BehaviorSubject` for a centralized and reactive data flow across the application.

### 🎨 Premium UI/UX
- **Modern Aesthetics**: Sleek dark/light mode friendly design with vibrant gradients and smooth transitions.
- **Responsive Layout**: Optimized for all screen sizes, from mobile to desktop.

## 🚀 Getting Started

### Prerequisites
- Node.js (v16+)
- Angular CLI (`npm install -g @angular/cli`)

### Installation

1. **Clone the Repository**
   ```bash
   git clone https://github.com/Kishor1407/user_dashobard.git
   cd user_dashobard
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Run Development Server**
   ```bash
   ng serve
   ```
   Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.

---

## 📂 Project Structure

- `src/app/dashboard`: Contains the main dashboard visualization and user list.
- `src/app/user-form`: Dedicated component for adding and editing users.
- `src/app/services`: `UserService` manages the application state and persistence logic.
- `src/styles.css`: Global styles and design system tokens.
